#Random Password Generator

Write a bash script which will fulfill the below requirements:

- Create a script which will ask the user to enter the length of password required.

- Then Script will validate the length provided by the user , it must be between 8 and 32 , else script must display error on screen.

- Then using /dev/urandom , design a random password for the user.


#!/bin/bash


echo -n "Enter length of password required : "

read num


if [[ $num -lt 8 ]] || [[ $num -gt 32 ]]

then

echo "Password length must be between 8 and 32"

else

echo "*******************************************"

echo "Here is your random password of length $numi : `tr -dc A-Za-z0-9 < /dev/urandom | head -c $num`"

fi
